\doxysubsection{详细描述}
B+ tree implementation. 

The multiset/multimap/set/map adapter has not been implemented yet. Only use when key is trivial. Otherwise, this maybe won\textquotesingle{}t faster than std\+::map because its search complexity is O((k -\/ 1) \texorpdfstring{$\ast$}{*} (log n / log k)). SSO optimization for tree node is enabled now!
