\doxysubsection{详细描述}
A unique stack allocator for fast simulation of stack memory on the heap. 

When a \doxylink{classwjr_1_1unique__stack__allocator}{unique\+\_\+stack\+\_\+allocator} object is destroyed, all the memory it allocates is released.~\newline
 And a new \doxylink{classwjr_1_1unique__stack__allocator}{unique\+\_\+stack\+\_\+allocator} constructed in the lifetime of a \doxylink{classwjr_1_1unique__stack__allocator}{unique\+\_\+stack\+\_\+allocator} object must be destroyed in the current lifetime.
