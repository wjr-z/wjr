详细描述

template<typename T>
class wjr::weak_stack_allocator< T >

Point to unique_stack_allocator.

Used for container. This allocator won't deallocate memory allocated by __small_allocate until container is destroyed.

Use a pointer to unique_stack_allocator to allocate memory. This class must be used carefully. If recursively using it as a reference and allocating memory, unique_stack_allocator should be avoided from being reused in the current function.
