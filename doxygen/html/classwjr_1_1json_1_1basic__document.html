详细描述

template<typename Traits>
class wjr::json::basic_document< Traits >

At present, it's a simple but flexible implementation solution. This is not as good as a only iteration in terms of performance.
Performance (use std::map and wjr::vector as container on my local x64 2.50 GHz machine) :
twitter.document : 700~720 MB/s
canada.document : 660~680 MB/s

待办事项
  1. For most strings, only need to check first character to compare.
  2. Maybe use hash like std::unordered_map, but for long strings, compare even faster than hash.
  3. In place construct without using low performance basic_document.
