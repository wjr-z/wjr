详细描述

A unique stack allocator for fast simulation of stack memory on the heap.

When a unique_stack_allocator object is destroyed, all the memory it allocates is released.
And a new unique_stack_allocator constructed in the lifetime of a unique_stack_allocator object must be destroyed in the current lifetime.
