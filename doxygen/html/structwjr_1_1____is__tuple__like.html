template<typename LP, typename RP>
struct wjr::__is_tuple_like< LP, RP >

Use template<...>typename like to like all element of LP and RP.

For example, like is std::is_assignable, LP is std::tuple<T0, U0>, RP is std::tuple<T1, U1>.
Then __is_tuple_like = std::conjunction<std::is_assignable<T0, T1>, std::is_assignable<U0, U1>>.
