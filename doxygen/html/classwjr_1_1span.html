详细描述

template<typename T, size_t Extent>
class wjr::span< T, Extent >

A view over a contiguous sequence of objectsd.

模板参数
Extentif Extent is dynamic_extent, the span is a runtime-sized view. Otherwise, the span is a compile-time-sized view.

Construct a span from a container.

The container must have a data() member function that returns a __is_span_iterator. The container must also have a size() member function that can be converted to size_type.
