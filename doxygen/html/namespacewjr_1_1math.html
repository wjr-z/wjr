noexcept

Performs addition with carry-in and carry-out, optimized for subsequent branching based on carry-out.

This function, addc_cc, adds two numbers with a carry-in, and returns the result and a carry-out. The carry-out (c_out) is optimized for subsequent code that branches based on its value. For example, it can be used with jump instructions like je or jne. This is in contrast to the addc function, which may use instructions like setc or test for branching.

注解
The carry-in and carry-out are limited to 0 and 1
模板参数
UType of the carry-in and carry-out. It must be an unsigned integral type.
参数
[in]c_inThe carry-in flag.
[out]c_outThe carry-out flag.

WJR_INTRINSIC_INLINE uint64_t wjr::math::asm_addc ( uint64_t a,
uint64_t b,
U c_in,
U & c_out )
noexcept

Use inline assembly to add two 64-bit integers with carry-in and return the carry-out.

The carry-in and carry-out flags are both 0 or 1.
The carry-out flag is set to 1 if the result overflows.
Optimization:

  1. Use constraint "i" if a or b is a constant and is in i32 range.
  2. If c_in is a constant and c_in == 1, use "stc" to set the carry flag.
模板参数
UThe type of the carry.
参数
[in]c_inThe carry-in flag.
[out]c_outThe carry-out flag.
返回
a + b + c_in
