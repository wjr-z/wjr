Assertion utilities

作者
wjr

WJR_DEBUG_LEVEL : 0 ~ 3
0 : Release
1 : Beta
2 : Runtime detect
3 : Maximize runtime detect, for debug
If WJR_DEBUG_LEVEL is not defined,
If NDEBUG is defined, WJR_DEBUG_LEVEL is set to 0 by default.
Otherwise, WJR_DEBUG_LEVEL is set to 1 by default.
WJR_ASSERT_L(level, expr) : Specify the level of assertion,
if the WJR_DEBUG_LEVEL is greater than or equal to the level,
the assertion is executed.
WJR_ASSERT(expr) : Equivalent to WJR_ASSERT_L(1, expr)
WJR_ASSERT_0(expr) : Always execute the assertion
